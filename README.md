### [📁 GitHub Issues Automation Script](https://github.com/iliy4s/shell-scripting)

![Bash](https://img.shields.io/badge/Bash-4EAA25?style=flat-square&logo=gnubash&logoColor=white)
![GitHub API](https://img.shields.io/badge/GitHub%20REST%20API-171515?style=flat-square&logo=github&logoColor=white)
![curl](https://img.shields.io/badge/curl-073551?style=flat-square&logo=curl&logoColor=white)

```bash
#!/bin/bash
curl -s -H "Authorization: token $PAT" \
     "https://api.github.com/repos/$OWNER/$REPO/issues" \
     | jq '.[] | {id: .number, title: .title, state: .state}'
```

> Built a CLI tool to fetch, filter and display GitHub issues using the REST API with personal access token auth and dynamic endpoints.

| 🎯 Key Win | Full API auth + JSON parsing · Zero external dependencies |
